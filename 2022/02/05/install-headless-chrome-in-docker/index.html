з”ЁDockerе®үиЈ…Headless Chrome

еҸ‘иЎЁдәҺ 2022-02-05

жңҖиҝ‘з”ЁHeadless ChromeжөҸи§ҲеҷЁй…ҚеҗҲSeleniumжҗһдәҶдёӢеүҚз«ҜйЎ№зӣ®зҡ„жөӢиҜ•пјҢж•ҲжһңиҝҳеҸҜд»ҘпјҢеңЁJenkinsйғЁзҪІд»Јз ҒеҗҺеј•е…ҘжөӢиҜ•пјҢиө·з ҒдҝқйҡңдәҶеүҚз«ҜйғЁзҪІеҗҺзҡ„еҸҜз”ЁжҖ§гҖӮиҝҷзҜҮж–ҮжЎЈж”ҫдёҠеңЁUbuntu 20.04зҺҜеўғдёӢе®үиЈ…ж— еӨҙChromeжөҸи§ҲеҷЁзҡ„DockerfileпјҢдҫҝдәҺд»ҘеҗҺжҹҘиҜўгҖӮ

DockerfileеҶ…е®№еҰӮдёӢпјҡ

и§ЈйҮҠдёӢе…¶дёӯзҡ„дёҖдәӣзӮ№пјҡ

  • aptй•ңеғҸжҚўжҲҗдәҶжё…еҚҺзҡ„жәҗпјҢе®үиЈ…дҫқиө–иғҪеҝ«зӮ№пјӣ
  • и®ҫе®ҡDEBIAN_FRONTENDдёәnoninteractiveпјҢеӣ дёәе®үиЈ…иҝҮзЁӢдёӯдјҡж¶үеҸҠеҲ°е®үиЈ…tzdataиҝҷдёӘеҢ…пјҢиҝҷдёӘеҢ…дјҡеҗҜеҠЁдәӨдә’иҜўй—®пјҢжңҖз»ҲеҜјиҮҙdocker buildзҡ„ж—¶еҖҷе°ұеҚЎдҪҸдәҶпјҢжүҖд»ҘйңҖиҰҒжҸҗеүҚи®ҫе®ҡдёӢиҝҷдёӘзҺҜеўғеҸҳйҮҸгҖӮдҪҶиҝҷж ·еҒҡзҡ„иҜқпјҢжңҖеҗҺдёҖжӯҘд№ҹе°ұйңҖиҰҒеҠ дәҶдёӘи®ҫзҪ®ж—¶еҢәзҡ„ж“ҚдҪңпјҢжүӢеҠЁжқҘи®ҫзҪ®дёӢж—¶еҢәпјӣ
  • е®үиЈ…дәҶдёӢж–Үжіүй©ҝзҡ„еӯ—дҪ“пјҢиҷҪз„¶еҸҜиғҪз”ЁдёҚеҲ°пјҢдҪҶеҸҜд»Ҙи§ЈеҶіеҜјеҮәpdfзҡ„ж—¶еҖҷдёӯж–Үж— жі•жҳҫзӨәзҡ„й—®йўҳпјӣ
  • е®үиЈ…chromeзҡ„ж—¶еҖҷе…Ҳз”Ё dpkg -i е‘Ҫд»Өе®үиЈ…дәҶдёҖж¬ЎпјҢиӮҜе®ҡдјҡеӨұиҙҘпјҢеӣ дёәзјәдҫқиө–пјҢжӯӨж—¶иҰҒз”ЁвҖҷexit 0вҖҷжқҘз»“е°ҫпјҢйҳІжӯўbuildе‘Ҫд»ӨйҖҖеҮәпјӣ
  • з”Ёapt install -f -yе°қиҜ•дҝ®еӨҚдҫқиө–пјҢеҸ–е·§пјҢжҳҜеҸҜд»Ҙе®үиЈ…жҲҗеҠҹзҡ„~
